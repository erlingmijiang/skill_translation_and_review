# skill_translation_and_review - 智能体技能翻译与安全审查工具

一个基于大语言模型的自动化工具，专为 Agent Skills 技能文件提供中文化翻译和安全审查分析功能，帮助中国AI用户和开发者快速理解和安全使用开源智能体技能。

## ✨ 功能特性

### 📝 智能翻译
- 智能识别代码结构：仅翻译注释、文档字符串、提示词等文本内容，不修改代码逻辑、变量名、函数名等核心标识
- 专业技术术语库：确保翻译准确符合中文开发者习惯，保留专有技术名词
- 语法安全校验：翻译后自动保持代码语法正确性，避免引入语法错误

### 🔒 安全审查
- 多层级风险检测：识别数据窃取、恶意操作、提示注入、广告植入等安全风险
- 风险分级机制：
  - 🔵 1级：完全安全，可正常使用
  - 🟡 2级：存在疑似风险，建议人工审核
  - 🔴 3级：存在严重安全风险，禁止使用
- 详细风险报告：每个风险点提供具体原因和位置信息
- 自动导出CSV：所有审查结果自动保存为CSV文件，便于批量管理

### ⚡ 使用体验
- 批量处理：支持整个目录递归扫描处理
- 进度显示：集成 tqdm 进度条，实时显示处理进度
- 日志记录：所有操作自动记录到日志文件，便于追溯
- 错误处理：完善的异常捕获机制，单个文件错误不影响整体流程

## 🚀 快速开始

### 环境要求
- Python >= 3.12
- uv 包管理器
- 支持 OpenAI 协议的大语言模型 API（推荐 DeepSeek、Claude、GPT-4 等）

### 安装步骤

1. **克隆项目**
```bash
git clone https://github.com/erlingmijiang/skill_translation_and_review.git
cd skill_translation_and_review
```

2. **安装依赖**
```bash
uv sync
```

3. **配置环境变量**
```bash
copy env.example .env
```
编辑 `.env` 文件，填入你的 LLM API 配置：
```env
LLM_API_KEY=你的API密钥
LLM_BASE_URL=模型API地址
LLM_MODEL_NAME=模型名
```

### 运行程序
```bash
uv run python main.py
```
程序会提示输入要处理的目录路径，输入后会自动开始批量翻译和审查。

### 翻译流程
1. 读取文件原始内容
2. 调用 LLM 按照翻译规则进行本地化处理
3. 验证翻译后代码语法正确性
4. 覆盖写入原文件（**注意：会修改原文件，请提前备份**）

### 审查流程
1. 读取翻译后的文件内容
2. 调用 LLM 进行静态安全分析
3. 检测潜在安全风险并分级
4. 结果存入审计列表，最终导出为 CSV 文件

## 🛡️ 安全提示

1. **审查验证：对于标记为 2 级和 3 级风险的文件，请务必进行人工复核**
2. **文件备份**：使用前请务必备份原始文件，程序会直接修改原文件内容
3. **API 安全**：请勿将 `.env` 文件提交到版本控制系统，防止 API 密钥泄露
4. **网络隔离**：审查过程不会主动访问任何外部链接，所有分析均为静态分析

## 🆘 常见问题

**Q: 翻译会破坏代码功能吗？**
A: 翻译提示词明确要求保留所有代码标识符、逻辑结构和语法，只会翻译注释和文本内容，不会影响代码功能。

**Q: 支持哪些大语言模型？**
A: 所有支持 OpenAI 协议的模型都可以使用，包括但不限于 DeepSeek、GPT-4、Claude、Qwen 等。

**Q: 可以只使用翻译功能或只使用审查功能吗？**
A: 当前版本是翻译+审查绑定流程，后续版本会支持独立运行单个功能。

**Q: 处理大量文件时速度慢怎么办？**
A: 速度取决于模型 API 的响应速度，可以考虑更换更快的模型服务，或调整并发数（需要修改代码）。
